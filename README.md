# OctoBoot (WIP !!)
OctoBoot is a fast, powerful, and user friendly tools to create your own website in a few clic, all free !! (with help of some free/custom template and gh-pages)

## Install

    > npm i

## Build
	
	First time
    > npm run build:semantic && npm run build

    Next time
    > npm run build

    Or watch
    > npm run watch
